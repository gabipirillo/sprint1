PROJETO

STORYTELLING

GERADOR DE POWER POINT

Somos o grupo Magnus, estamos no 3º ano de Engenharia de computação da FIAP, este é um projeto da FIAP em parceria com a empresa Sanofi, a qual nos deu inúmeras propostas de projetos para escolhermos um e prosseguir. Escolhemos a proposta de Storytelling.

PROPOSTA SANOFI

“A área gasta muito tempo preparando apresentações e vídeos para resumir os projetos e atividades do ano anterior para as áreas de negócio em reuniões curtas de até uma hora. Existe a necessidade de criação de apresentações e conteúdos com base nos dados de qualidade com a abordagem de storytelling.”

SOLUÇÃO ENCONTRADA

“Iremos criar um programa de IA generativa. Pegaremos como base a estrutura do Gemini Google IA Studio, que é um software de Inteligência Artificial, onde você consegue importar arquivos e ele te faz um resumo otimizado do documento. Para o conteúdo da apresentação, utilizaremos o banco de dados da empresa, sem as informações técnicas, com os dados dos últimos 365 dias. Já para o visual estético, nos basearemos no site Gamma App, que é um software de IA que gera apresentações de Powerpoint com o tema em pauta que o usuário escolher. Treinaremos a IA para criar o storytelling de acordo com o banco de dados da empresa de uma forma que cative e prenda a atenção dos clientes.”
